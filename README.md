## 🎭 What is OpenMythos?
**OpenMythos** is a theoretical reconstruction of the *Claude Mythos* architecture. It implements a **Recurrent-Depth Transformer (RDT)** that allows AI to "think deeper" by looping through internal layers without increasing parameter count.

This repository features the **Ultimate Neural Dashboard**, a high-end web interface designed to visualize the internal "thinking" process of the Mythos model in real-time.

---

## 🚀 Key Features (v2.0 PRO)

- 🧠 **Brain Stream Visualization**: Live line charts showing activation levels across each recurrence loop.
- ⚡ **Thinking Depth Toggles**: Switch between `Fast` (1 loop), `Deep` (16 loops), and `Philosophical` (64 loops) thinking modes.
- 🧪 **Stability Guardrail**: Real-time *Spectral Radius* monitoring to ensure neural stability.
- 🤖 **MoE Expert Map**: Visual indicators for Mixture of Experts (MoE) firing patterns.
- 🌌 **Neural Particle UI**: A stunning Sci-Fi interface with particle backgrounds and glassmorphism.
- 🔊 **Neural SFX**: Immersive sound effects for a complete "Command Center" experience.
- 📄 **Data Export**: One-click PDF export for research documentation.

## 🛠️ Quick Start

### 1. Requirements
Make sure you have Python 3.11+ and the following dependencies:
```bash
pip install flask torch transformers open-mythos
```

### 2. Launch the Command Center
Clone this repository and run the dashboard:
```bash
git clone https://github.com/amirudinR/openmythos.git
cd openmythos
python app.py
```
Open your browser and swim into the deep at: `http://127.0.0.1:5000`
